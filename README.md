# CSS Only Sidebar Navigation

A responsive sidebar navigation menu built using **HTML5** and **CSS3** only, without JavaScript. This project demonstrates how to create an interactive hamburger menu and sliding sidebar using the CSS checkbox hack.

## Tech Stack

* HTML5
* CSS3

## What Makes This Project Special
This project features an interactive sidebar navigation menu built entirely with HTML and CSS, using the CSS checkbox hack with :checked and sibling selectors to create JavaScript-free interactivity.

The most interesting part is the hamburger menu functionality, where clicking the menu icon opens and closes the sidebar using a hidden checkbox technique. This demonstrates how CSS can be used not only for styling but also for creating interactive UI behavior.

The project also includes smooth sidebar animations, hover effects, icon transitions, and a modern glassmorphism-inspired design, making the interface visually appealing and user-friendly.

It is a great example of advanced CSS concepts such as positioning, transitions, pseudo-classes, and responsive layout techniques.

This project helped strengthen my understanding of building functional and interactive web components with clean frontend code.
